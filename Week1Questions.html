Website Structure Question 1

How are heading elements similar and different from the header element?

Heading elements h1-h6 give an indication of what the following content is about and hence "introduce" that content. The header element is similar in this regard, however the header element is a container and can contain multiple elements. In addition it is good and commont practice to include a heading element within a header element.
